# 🏠 FixMyHostel

A secure and anonymous hostel complaint management system designed to help students report issues safely while enabling hostel authorities to manage, track, and resolve complaints efficiently.

---

# 🌟 Overview

FixMyHostel is a full-stack web application developed to improve communication between hostel residents and hostel management. The platform allows students to anonymously report complaints such as maintenance issues, bullying, harassment, safety concerns, and emergencies.

The system ensures that complaints are routed to the appropriate authorities while maintaining student privacy and providing real-time complaint tracking.
